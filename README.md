# Loan Approval Predictor Using Decision Trees

A Machine Learning project that predicts whether a loan application will be approved or rejected based on applicant information such as income, education, employment status, credit history, loan amount, and other relevant features.

The project uses the **Decision Tree Classification Algorithm** to build an interpretable predictive model for loan approval decisions.

---

## 📌 Project Overview

Financial institutions receive thousands of loan applications every day. Manually evaluating each application can be time-consuming and prone to inconsistencies.

This project aims to automate the loan approval process by:

- Analyzing applicant data
- Training a Decision Tree Classifier
- Predicting loan approval status
- Evaluating model performance using standard metrics

---

## 🚀 Features

- Data preprocessing and cleaning
- Handling missing values
- Feature encoding
- Exploratory Data Analysis (EDA)
- Decision Tree model training
- Model evaluation
- Loan approval prediction
- Easy-to-understand decision tree visualization
